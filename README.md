# ipfs-sprint-helper

> An IRC bot to help IPFS announce its sprints

## Install

```sh
> npm i ipfs-sprint-helper
```

## Usage

```sh
> node index.js
```

In IRC:

```
> sprint-helper: "Title name" 827 http://hackmd.io http://zoom.us/call/21880801 https://www.youtube.com/watch?v=ZZ5LpwO-An4
> sprint-helper: next
```
